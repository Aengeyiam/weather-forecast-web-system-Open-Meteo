| 属性 | 内容 |
|------|------|

---

## 二、启动方式

**纯前端（最快）：** 双击 `index.html`，浏览器打开即用。数据存 localStorage。

**PHP + MySQL 完整版：**
1. 安装 XAMPP，启动 Apache + MySQL
2. 复制整个 `weather-system` 文件夹到 `C:\xampp\htdocs\`
3. 打开 http://localhost/phpmyadmin → 导入 `sql/init.sql`
4. 访问 http://localhost/weather-system/

## 四、项目文件结构

## 六、技术架构

| 层级 | 技术 | 说明 |
|------|------|------|
| 页面 | HTML5 | SPA单页，CSS切换登录/主页 |
| 样式 | CSS3 | 蓝色主题，卡片式布局，动画 |
| 逻辑 | Vanilla JS | 模块化，零第三方库 |
| 后端 | PHP PDO | login/register/history接口 |
| 数据库 | MySQL | InnoDB, utf8mb4, 2张表 |
| 加密 | SHA-256 | 浏览器crypto + PHP hash |
| 天气 | Open-Meteo | 免费，无需Key |
